**relevant keys for this yazi config**

*help*

- `F1` - this page
- `SHIFT-F1` - show all mappings
- `e` - execute commands using `fzf`

*operations*

- `q` - quit
- **`ENTER` - file type sensitive operations**
- `SPACE` - select / unselect files
- `y` - yank / copy menu
  - `y` `y` - copy files
- `f` - find / search menu
  - `f` `f` - fuzzy search (fzf)
- `z` - filter (in current folder)
- `o` - order menu
  - `o` `n` - order natural (default)
- `g` - goto menu
  - `g` `h` - goto home
  - `g` `SPACE`- interactive
  - `g` `z` - go using zoxide
- **`k` - custom menu**
- `tab` - toggle file information
- `DELETE` - delete files to trash
- `Shift + DELETE` - delete files permanently
- `F2` - rename

*navigation*

- `m` `n` - move up / down in preview
- `v` `b` - move left / right in preview (tables)
- `F11` - toggle preview fullscreen
- `F10` - toggle 2 panes view
- `√∂` `√§` - previous / next folder in history
